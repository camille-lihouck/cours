sorting.merge(t1, t2, cmp)[source]¶

Given two sorted list, creates a fresh sorted list.

Parameters:
  • t1 (list) – A list of objects
  • t2 – A list of objects
  • cmp (function) – A comparison function, returning 0 if a == b, -1 is a < b, 1 if a > b
Returns:

A fresh list, sorted.
Return type:

list

Note

time complexity of merge is \(O(n_1+n_2)\) with \(n_1\) and \(n_2\) resp. the length of t1 and t2

sorting.partition(s, cmp)[source]¶

Creates two slices from s by selecting in the first slice all elements being less than the pivot and in the second one all other elements.

Parameters:
  • s (dict) – A slice of a list, that is a dictionary with 3 fields : data, left, right representing resp. a list of ibjects and left and right bounds of the slice.
  • cmp (function) – A comparison function, returning 0 if a == b, -1 is a < b, 1 if a > b
Returns:

A couple of slices, the pivot being at the left index of the second slice.
Return type:

tuple
